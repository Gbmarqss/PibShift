# Pib Shift

Pib Shift é uma aplicação de desktop para gerar escalas de trabalho para os ministérios da igreja.

## Funcionalidades

- Gera escalas a partir de um arquivo Excel com a disponibilidade dos voluntários.
- Permite a seleção de áreas (ministérios) para a escala.
- Visualização da escala gerada em formato de cards.
- Configurações personalizáveis, como diretório padrão para salvar os arquivos.

## Como usar

1.  Execute o arquivo `pibshift.py`.
2.  Na aba "Gerar Escala", selecione o arquivo Excel com a disponibilidade.
3.  Selecione as áreas desejadas.
4.  Clique em "Gerar Escala".
5.  A escala gerada será exibida na aba "Visualizar Escala" e salva como um arquivo Excel.
